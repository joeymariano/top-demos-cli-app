Top Demos CLI App
======
**Top Demos CLI App** is a command line app. It uses the nokogiri library to crawl pouet.net and return youtube links of the top computer demos.

## Usage
* git clone
* cd top-demos-cli-app
* bundle
* ./bin/top-demos (on Linux)
